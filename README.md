## Install

1. Download the APK on your Android device
2. Open the file from your downloads folder
3. If prompted, enable "Install from unknown sources" for your browser
4. Tap **Install**

Requires **Android 6.0 (Marshmallow) or higher**.

### Realme / Oppo / ColorOS users

You may need to enable two extra toggles in Developer Options:

1. *Settings → About phone → tap "Build number" 7 times* (enables Developer Options)
2. *Settings → Additional / System Settings → Developer Options →*
   - **Install via USB** → ON
   - **Disable Permission Monitoring** → ON (optional)

Without these the system blocks installs from outside the Play Store.

## What is Purohit?

An offline-first Android app for Hindu purohits to manage yajmaans,
poojas, dakshina, samagri, and devansh. Built by
[Ramarpan Creations LLP](https://purohit.rpit.in) (RPIT Computers),
Mumbai.

### Key features

- Yajmaan + booking management with per-booking reminders
- Post-pooja wizard (दक्षिणा · देवांश · सामग्री · धन्यवाद)
- Income / expense ledger with auto-derived rows from completed poojas
- **दृक् पंचांग** — 30-day offline cache (tithi, nakshatra, yoga, karana,
  sunrise, sunset, rahukaal, abhijit muhurat)
- Pre-pooja notifications (1 day + 2 hours before)
- AES-256-GCM encrypted local backup with user password
- PIN + biometric app lock
- 12 Indian languages (Hindi, English, Marathi, Gujarati, Sanskrit,
  Bengali, Tamil, Telugu, Kannada, Malayalam, Punjabi, Odia)

### Privacy

Your data stays entirely on your device. No account, no internet
dependency, no telemetry. See the [privacy policy](https://purohit.rpit.in/privacy.html).
